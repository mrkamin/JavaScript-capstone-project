## 💻 Getting Started <a name="getting-started"></a>

### Prerequisites

In order to run this project you need: a properly working pc or laptop and a modern web browser.

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone git@github.com:mrkamin/JavaScript-capstone-project.git
```

### Install

Install this project with:

```sh
  cd my-folder
  git clone git@github.com:mrkamin/JavaScript-capstone-project.git
  npm install
```

### Usage

Use this app by runing:

```sh
  cd my-folder
  git@github.com:mrkamin/JavaScript-capstone-project.git
  npm install
  npm start
```

### Run tests

Test this app by runing:

```sh
  cd my-folder
  git git@github.com:mrkamin/JavaScript-capstone-project.git
  npm install
  npm test
```

You can deploy this project using: GitHub but if you want to deploy make sure you let me know first :3
